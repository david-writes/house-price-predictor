# 🏠 House Price Predictor
### AI & ML Internship — Task 1 | Maincrafts Technology

## 📌 Overview
A Linear Regression model that predicts median house prices 
across California neighborhoods using the California Housing dataset.

## 📊 Dataset
- Source: California Housing (scikit-learn built-in)
- Records: 20,640
- Features: 8
- Target: Median House Value

## 🤖 Model Results
| Metric | Value |
|--------|-------|
| MAE    | $53,320 |
| RMSE   | $74,560 |
| R²     | 0.576   |

## 🛠️ Tools Used
- Python 3.11
- pandas, numpy
- scikit-learn
- matplotlib, seaborn
- Jupyter Notebook
